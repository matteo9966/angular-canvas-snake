## Tutorial
Choose 1 or 2 players 

![image](https://github.com/matteo9966/angular-canvas-snake/assets/75813215/c9a779da-679d-479d-b483-b30e065de1da)


commands for the players are

### PLAYER 1 
|KEYBOARD KEY | SNAKE DIRECTION|
|-------------|---------------|
| **W** | ↑ |
| **S** | ↓ |
| **D**  | → |
| **A** | ← |

### PLAYER 2 

|KEYBOARD KEY | SNAKE DIRECTION|
|-------------|---------------|
| **ARROW UP** | ↑ |
| **ARROW DOWN** | ↓ |
| **ARROW RIGHT**  | → |
| **ARROW LEFT** | ← |
